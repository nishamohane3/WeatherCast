# WeatherCast

Weather forecasting app according to current location

## Installation

1. Clone the repository.
2. Install dependencies using `npm install`.
3. Run the index.html file.

## Description

The application WeatherCast; our weather app will show the weather information that it will get from the API, an Icon that shows the weather status, the temperature value (18Â°C) in Celsius unit, the weather description (clear sky), and then the user's city and country.

When the user clicks on the temperature value, that will convert it from Celsius to Fahrenheit.

The API provider is http://www.OpenWeatherMap.org
